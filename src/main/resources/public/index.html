How to

The application allows to create a mapping aka mock endpoint using the above form. This can be later viewed, updated or deleted. Links for these actions can be found in the top menu.

Mocker

The Mocker web application behaves like a mock server that allows the user to create
mappings aka mock endpoints and define possible responses for the configured requests.
