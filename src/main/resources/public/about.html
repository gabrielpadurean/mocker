About

The Mocker web application behaves like a mock server that allows the user to create mappings that define possible HTTP responses for the configured HTTP requests.

How to

The application allows to create a mapping (between an HTTP request and an HTTP response) using the above form. This can be later viewed, updated or deleted. See links in the top menu.
When an HTTP request is made to the application and matches the details from the Request section then an HTTP response is sent back containing the details from the Response section.

The necessary steps:
- create a mapping (filling the form above) for an endpoint or URL that you need
- make an HTTP request to the endpoint or URL specified (should match the details entered in the Request section)
- the HTTP response will contain the details entered in the Response section
- the mapping can be updated or deleted (optional)
